# The Adventurer's Guild

A fantasy-themed website created as part of a Web Design course. The site serves as a fictional information hub for The Adventurer's Guild, an organization dedicated to connecting adventurers with quests, training, resources, and opportunities throughout the realm.

## Project Overview

The Adventurer's Guild website is designed to simulate a fully functioning guild organization within a fantasy setting. Visitors can learn about the guild, explore its ranks and locations, browse quest boards, review resources, and follow guild news and announcements.

The project focuses on website organization, visual design, navigation structure, typography, responsive layouts, and user experience while maintaining a consistent fantasy-inspired theme.

## Features

* Fantasy-themed visual design
* Custom typography using web fonts
* Hero image slider with animated transitions
* Structured navigation system
* Organized multi-page website architecture
* Guild rank progression system
* Quest board system
* Guild locations and facilities
* Resource library
* News and announcements section
* Hidden easter egg achievements
* Responsive design principles

## Design Goals

The primary goals of this project are:

* Create a cohesive fantasy-themed website.
* Maintain consistent visual branding across all pages.
* Organize content in a scalable and maintainable structure.
* Demonstrate proficiency with HTML and CSS.
* Provide an engaging user experience through visual effects and interactive elements.
* Build a project that can continue to grow beyond the requirements of the course.

## Hidden Achievements

The website contains a number of hidden achievements inspired by tabletop roleplaying games. Curious visitors who thoroughly explore the site may discover special achievement cards hidden throughout the guild's records.
